Amazon affiliateの特長！あなたの好きな商品が全て揃う！

Amazon affiliateの大きな特長は次の3つ。

  1. 誰もが知っているECサイト
  2. あなたの得意なジャンルの商品が揃っている
  3. 報酬単価は低いが安定

では、具体的に見ていきましょう。

Amazonと言えば誰もが知っているECサイト！安心して購入できる！

Amazon affiliateの最大の特長と言えば、母体が巨大なECサイトという点です。

Amazonを利用した詐欺などのニュースもありましたが、それでも尚Amazonというだけで安心感が得られます。

あなたのサイト訪問者としても、Amazonで購入できるというだけで安心して購入できるのではないでしょうか。

特に書籍や電化製品などは実店舗で購入するより安く買えることもありますし、わざわざ出掛けて探すよりはAmazonで手軽に購入したいという人も多いに違いありません。

とにかく大きなECサイトなので、買えない物を探す方が難しいくらいです。

Amazon affiliateには得意ジャンルの商品が揃っている

アフィリエイトで大事なことは、あなたの得意なジャンルに取り組むということです。

しかし、あなたの得意なジャンルの広告がASPにはほとんど無いという場合も少なくありません。

そんな時はAmazon affiliateを利用して商材を紹介するという方法がおすすめです。

Amazon affiliateにはあらゆる商材が揃っていると言っても過言ではありません。ですから、他のASPに無いジャンルの関連商品でも容易に見付けることができるでしょう。

特にあなたが雑記ブログや趣味のブログなどを運営している場合には、

  • Amazon affiliate
  • Google アドセンス

を利用して稼ぐという方法が最適。

また、アドセンスではどの広告がクリックされてどれだけ報酬が得られたかというのがわかりませんが、Amazon affiliateの場合は訪問者が購入した商品を知ることができるので、サイトの方向性を決める場合にも役立ちます。

しかし、Amazon affiliateの報酬単価は低く設定されているものが多いのが残念なところです。

ただし、商材によっては意外と高額なものもあるので、その点についても見ていきましょう。

Amazon affiliateの報酬単価は安いが安定！狙うはAmazonプライム

Amazon affiliateの報酬単価は安いというイメージがありますが、意外と高額な商品もあります。

Amazon affiliateで紹介できる商品としては、大きく分けると次の2種類です。

  • Amazonプライム
  • Amazonプライム以外（主に物販）

一般的な商材としてはAmazonプライム以外となります。

報酬に関しては次の表をご覧ください。

報酬料率は0～10%。しかも、上限が1000円と決まっているので、例えば報酬料率2%の家電なら5万円以上の商品は全て1000円の報酬となってしまいます。

その点、Amazonプライムの報酬は設定が違うので狙い目です。

特におすすめは「無料体験」の申し込みで報酬が得られるプログラム。

無料体験なのに500円～3000円の報酬額となっています。特に「Audible」「Amazonビジネス」への登録は報酬額3000円となっているので狙い目です。

Amazon affiliateの特長まとめ
  1. 誰もが知っているECサイト
  2. あなたの得意なジャンルの商品が揃っている
  3. 報酬単価は低いが安定

では、Amazon affiliateはどのようにすれば利用ができるのでしょうか。

Amazon affiliateの登録時の注意！審査に通らない場合は裏技を！

Amazon affiliateの登録作業についてはそれ程難しくはありません。

しかし、少し前からAmazon affiliateの審査が非常に厳しくなっているので、申請をしてもなかなか審査が通らないという話を耳にするようになりました。

では、審査に通る為にはどのようにすれば良いのでしょうか。

Amazon affiliateの審査に受かる方法

まずはAmazonの公式見解を確認しましょう。

Amazon affiliateを利用する場合の公式基準としては次のようになっています。

すべてのサイトは(広告が削除されても)問題ないようなオリジナルのコンテンツが必要です。記事数の目処として約10件の記事をご用意ください。 審査対象のウェブサイトやアプリケーションは公開されており、アクセス可能である必要があります。閲覧に制約があったり、有料コンテンツがあるような場合は参加することができません。

公式見解としては、オリジナルの記事が10件程度あれば問題ないということです。

ですから、他の広告を掲載していない状態でAmazonの商品を掲載しやすいような記事を書いておくことをおすすめします。

それでもAmazon affiliateの審査に通らないという場合には以下の2点を試してから再度申請してみましょう。

  • Amazonプライムに申し込む（無料体験）
  • 記事内にAmazonの商品を掲載する
  • サイトに運営者情報・プライバシーポリシーを掲載する

Amazonとしてはお客さんであるあなたを蔑ろにすることはできないので、Amazonプライム会員の場合は審査の時に優遇されると言われています。

更に、記事内にAmazonの商品を掲載することで、Amazon側への印象は良くなることが期待できます。「報酬を得られない状態でも紹介しています」「私はAmazon大好きです」というアピール的に利用するのがおすすめです。

また、近年は運営者情報・プライバシーポリシーの掲載に関して非常に厳しくなっています。

ですから、Amazon affiliateに限らずASPに申請する時には必ず運営者情報とプライバシーポリシーのページを作成しておきましょう。

運営者情報・プライバシーポリシーの書き方がわからない場合は、ネットで検索すると解説しているページがたくさんありますので、例文をそのままコピーしてしまって問題ありません。

運営者情報に関しては個人情報を知られたくないという場合、画像にして貼り付けることをおすすめします。

しかし、それでもAmazon affiliateの審査に通らない場合には裏技を使いましょう。

Amazon商品は他のASPでもアフィリエイト可能！

実は、Amazon商品は公式のAmazon affiliate以外のASPでもアフィリエイトが可能です。

しかも、料率はAmazon affiliateと同じなので、わざわざ難しい審査に挑戦する必要はありません。

何度も審査に落ちるようなら、速やかに他のASP経由で提携申請をしてみましょう。

Amazonの商品をアフィリエイトできるASPは次の3社です。

  • A8ネット
  • もしもアフィリエイト
  • バリューコマース

まずはA8ネットの場合です。

A8に登録した時点で勝手に提携されるので、特に提携申請は不要です。

A8.netに登録する時にはサイトが無くても登録ができます。

つまり、A8.netを経由してAmazonのアフィリエイトをするにはサイトの審査が不要ということになります。

ただし、A8.netでAmazonの商品をアフィリエイトする注意点としては「アダルト商品」の紹介ができないという点です。

 

続いて、もしもアフィリエイトについて紹介しましょう。

こちらがもしもアフィリエイトの提携画面です。

もしもアフィリエイトを通じてAmazonの商品をアフィリエイトする場合は審査が必要と書かれています。

しかし、Amazon affiliateに直接申し込むことを考えると、もしもアフィリエイトの審査は非常に緩いようです。

 

最後にバリューコマースについて紹介しましょう。

上の画像がバリューコマースのAmazon提携画面となります。

バリューコマースの場合は、「コンテンツが不十分であり不明慮に成果が発生していた場合は成果対象外」と書かれていますが、平均承認率が98.77%となっていますので、ほとんどそのようなことは無いと考えて良いでしょう。

Amazon affiliateの登録時の注意

オリジナルの記事数10件程度を用意したサイトで申請する。
なかなか審査に通らない場合は、Amazonの商品をアフィリエイトできるASPを利用しよう。

Amazon affiliateのリンクの貼り方

Amazon affiliateと提携することができたら、早速あなたのサイトでAmazon商品を掲載して紹介してみましょう。

Amazon affiliateの商品リンクは以下の3種類から選べます。

  • テキスト
  • 画像
  • テキストと画像

また、URLを「短縮URL」に変更することも可能。短縮URLを使用すると、メールやSNSなどに掲載するときに非常に便利です。

Amazon affiliateの商品リンクを掲載する方法としては、アフィリエイトリンクを直接貼り付ける方法が一般的ですが、よく見かけるのが、Amazon・楽天市場・Yahooショッピングの3つの商品が掲載されているリンクです。

中にはAmazonよりも楽天やYahooショッピングの方が利用しやすいという人も居られるので、Amazon以外のアフィリエイトリンクを貼り付けることで取りこぼしがなくなるのでおすすめです。

方法としては次の2種類があります。

  • カエレバを利用する
  • ワードプレスのプラグインを使う

こちらがカエレバ。

そして、こちらがワードプレスのプラグイン「WP Associate Post R2」です。

どちらも無料で利用できるので、あなたのサイトに是非導入してみてください。

まとめ

本記事では、Amazon affiliateについて紹介しました。

Amazon affiliateの物販は報酬料率が低いことから、大きく稼ぐことが難しいと思われがちですが、アドセンスなどと上手く組み合わせることで成功することも十分可能。

特に、Amazon affiliateの商材はジャンルも多岐にわたる為、他のASPでは取り扱っていないニッチな分野でもアフィリエイトが可能です。

ただし、Amazon affiliateの登録時に審査がなかなか通らない場合があるので、その場合は「A8.net」「もしもアフィリエイト」「バリューコマース」などの他のASP経由で提携することも検討してみてください。
