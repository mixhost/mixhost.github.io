WordPressのデメリット

WordPressの主なデメリットは次の4点です。

  1. レンタルサーバーが必要
  2. 攻撃者から狙われやすい
  3. ページの読み込み速度が遅い
  4. 頻繁にシステムの更新をしなければならない

まず、WordPressはプログラムで動作しているので、WordPressをインストールするレンタルサーバーが必要となります。

レンタルサーバーは無料のサービスもありますが、おすすめできません。

しかし、WordPressの利用者が非常に多いのは、それ以上のメリットがあるからです。

WordPressのメリット

WordPressの主なメリットは次の5点です。

  1. プログラム自体は無料で利用できる
  2. 拡張性と自由度が高い
  3. デザイン性に富んでいる
  4. ブラウザ上で記事の作成・更新ができる
  5. WordPressに関する情報が多い

サーバーや独自ドメインを契約する必要はありますが、WordPressのシステム自体は無料で利用できるというのが最も大きなメリットでしょう。

また、WordPressのシステムだけでは足りない機能があったとしても、プラグインを利用することで機能を補うことも可能です。

WordPressは色々な機能を追加することができるので、不足していると感じた点はほとんどの場合プラグインで補うことが可能。

とにかくWordPressを利用すればどんなことでもできるので、どのブログシステムを使用するのかということで悩んだ場合はWordPressを選べば間違いないでしょう。

はてなブログの特徴！初心者でもカンタン！

はてなブログ

多くの方がWordPress意外のブログ作成サービスを選ぶなら「はてなブログ」だと言います。その理由としては次の通り。

  • ブログ初心者でも簡単に利用可能
  • SEOに強い
  • アフィリエイトが可能
  • 「はてなブックマーク」の利用が可能
  • 有料プランは独自ドメインが利用可能
  • サーバーが安定している

特にはてなブログをおすすめするのは「はてなブックマーク」という独自のコミュニティが出来上がっているという点です。

はてなブログを利用している他のブロガーとの繋がりができるので、立ち上げ初期からのアクセスも期待できます。

この点はWordPressには無いメリットなので、ブログ立ち上げ当初からアクセスが欲しいという場合にはおすすめ。

しかし、はてなブログには次のようなデメリットがあります。

  • カスタマイズ性が低い
  • デザイン性が低い
  • 無料プランは容量が少ない
  • 無料プランは広告が表示される
  • 無料プランは独自ドメインが利用できない

ですから、はてなブログを利用するなら無料プランではなく、有料プランに申し込むのがおすすめです。ただ、はてなブログの有料プランに申し込むと、1つのブログしか利用できません。

その点、WordPressはサーバーに複数のブログを構築することも可能です。

ですから、複数のブログを運営していきたいという場合には断然WordPressがおすすめです。

AmebaブログはSNS感覚で利用可能！

Amebaブログ

芸能人のブログと言えば「Amebaブログ」というイメージが強いのではないでしょうか。

あなたもご存知の通り、スポーツ選手や多くの著名人もAmebaブログを利用して情報発信されています。

では、Amebaブログのメリットとデメリットについて考えてみましょう。

Amebaブログの主なメリットとしては、次の3点。

  1. 無料で利用できる
  2. Amebaブログのユーザーが多い
  3. SNSに近い感覚で利用可能

Amebaブログはユーザー数も多く、足跡機能やフォロー機能などのSNSに近い機能もあるので、はてなブログ以上に横との繋がりい強いと言えます。

ですから、ブログ記事がそれ程面白い内容でなかったとしても、横との繋がりを重視することでアクセスを集めることは可能。

初動でアクセスを稼ぎたいという場合にはAmebaブログはおすすめです。

しかし、無料ということで、広告が表示されるのは仕方がありません。この問題も有料プランにすることで広告表示を無くすことも可能ですが、料金を支払ってまで利用する価値は感じられません。

また、カスタマイズは非常に困難なので、他のアメブロユーザーとの差別化をするのは難しいでしょう。

ライブドアブログはLINEで更新通知を受け取れる！

ライブドアブログ

「ライブドアブログ」のメリットとデメリットについて考えてみましょう。

ライブドアブログの主なメリットとしては次の4点です。

  1. 無料で利用できる
  2. ブログデザインが豊富
  3. LINEでブログ更新通知が受け取れる
  4. アダルトコンテンツ可

特にLINEで通知が届くというサービスについては他のブログ作成サービスではあまり見掛けない機能です。

ただし、お気に入りに登録する必要があるので、利用価値があるか否かはやや疑問です。

また、ライブドアブログはアダルトコンテンツも利用可能な数少ないブログ作成サービスなので、アダルトコンテンツを作成する場合にはおすすめのブログ作成サービスです。

では、ライブドアブログのデメリットはどうでしょうか。

  1. スマホ版はデザイン性が低い
  2. スマホ版の広告は削除不可能

特にスマホ版に関してはデメリットが大きいと感じます。

近年はパソコンよりもスマホからのアクセスが多いので、そういった点でもライブドアブログをおすすめすることはできません。

FC2ブログはカスタマイズ性が高い！

FC2ブログ

次に、「FC2ブログ」のメリットとデメリットについて見ていきましょう。

FC2のメリットとしては、次の通りです。

  1. カスタマイズ性が高い
  2. ユーザーが多い
  3. 各種ランキングによるアクセスが多い
  4. アダルト系も利用可能
  5. SEOに強い

FC2はランキングが非常にわかりやすく、そのランキングからのアクセスも期待できるという点ではおすすめです。

また、アダルト系の記事でも利用可能ですが、直接的な表現はNGのようです。

一方、デメリットは次の通り。

  1. アダルトや無法地帯のFC2動画により将来的に不安
  2. Bingサーチエンジンには表示されない
  3. 1ヶ月間ブログを更新しないと広告が表示される

アダルト系や無法地帯の動画もFC2にあるということも関係しているのか、BingサーチエンジンではFC2ブログが表示されにくい状況になっています。

現状ではSEOに強い印象のFC2ですが、今後はBingだけではなくGoogleサーチエンジンにも影響があるかもしれません。

ですから、長期的な維持を考えるとFC2はおすすめできないブログと言えるでしょう。

Seesaaブログは無料プランでも独自ドメインが利用可能！

Seesaaブログ

「Seesaaブログ」の主なメリットは次の3点です。

  1. 無料プランでも独自ドメインの利用可能
  2. 複数ブログが作成可能
  3. 容量が5GB

特に無料プランでも独自ドメインが利用可能というのは大きなメリットです。

また、1つのアカウントで最大5つまでブログを作成可能なので、いくつかのブログを作成したいという場合にはSeesaaがおすすめです。

他のブログ作成サービスの場合は1つずつアカウントを作る必要がありますが、Seesaaの場合はその手間が省けます。

では、デメリットはどうでしょうか。Seesaaブログのデメリットは次の3点です。

  1. 管理画面が重い
  2. スマホ用ページの広告は消せない
  3. SEOに弱い印象

管理画面が重い印象がありますが、慣れればそれ程問題ではありません。また、利用環境などによっても左右される印象です。

他のブログ作成サービスでも同様な場合がありますが、無料プランの場合はスマホ用ページの広告が削除不可となっています。

ですから、Seesaaブログは商用利用は可能と言ってもアフィリエイト目的で利用するのはおすすめできません。

JUGEMブログはアフィリエイト可！

JUGEMブログ

「JUGEMブログ」はインターネット事業を広く手掛けているGMOが運営しています。

ロリポップのレンタルサーバーを契約すると、手軽に利用できるということでロリポップ契約を機に利用しているユーザーも多いようです。

では次にJUGEMブログのメリットとデメリットについて見ていきましょう。

JUGEMブログの主なメリットは次の4点です。

  1. アフィリエイト可
  2. ブログが削除されにくい
  3. 有料プランが月額300円
  4. 各種SNSとの連携も可能

近年はアフィリエイト可のブログ作成サービスが非常に増えてきましたが、ブログ作成サービスによってはアフィリエイト記事ばかりを投稿しているとアカウントを凍結される可能性もあります。

しかし、JUGEMブログの場合はその心配は要りません。もちろんアダルトコンテンツはNGですが、一般的なアフィリエイト記事を投稿してもブログが削除されることはありません。

では、JUGEMブログのデメリットについて見ていきましょう。

  1. デザイン性が低い
  2. 画像容量は無料プランで最大1GBまで

JUGEMブログのデザインはテンプレートで簡単に変更できますが、自分でカスタムするのは少し難しいという印象です。

同様のブログ作成サービスのカスタマイズ性という点ではFC2に及びません。

また、画像容量は無料プランで最大1GBまでとなっているので、多くの画像をブログに掲載することは不可能です。

有料プラン「JUGEM PLUS」では無制限ですが、無料プランは長期的な利用という点ではおすすめできないブログ作成サービスと言えます。

gooブログ

「gooブログ」はNTTグループの運営する無料ブログ作成サービスです。

管理画面が非常にわかりやすく、初心者でも簡単に利用できるというメリットがあります。

しかし、主なメリットはそれだけで、デメリットの方が多い印象です。gooブログの主なデメリットは次の3点。

  1. デザインのカスタマイズ性が低い
  2. サイドバーも変更不可能
  3. 無料プランはアフィリエイト不可

gooブログはカスタマイズ性が低いというよりほとんど変更不可能なブログ作成サービスです。

ですから、すぐにブログを開始したいという場合にはおすすめしますが、初心者の場合を除いておすすめできるブログ作成サービスではありません。

楽天アフィリエイトを始めるなら楽天ブログ！

楽天ブログ

  2. 写真・動画の登録が簡単
  3. 足跡機能
  4. 掲示板が利用できる

楽天ブログのおすすめできる点として、楽天アフィリエイトと親和性が高いという点です。

楽天ブログ開設と同時に楽天アフィリエイトを始めることができ、購読者は楽天会員がメインなのでアフィリエイト成果も発生しやすいのも特長です。

楽天アフィリエイトと親和性が高いことをメリットとして挙げましたが、実は楽天アフィリエイト以外のアフィリエイトはできません。

あなたの記事を収益化するならnoteがおすすめ！

note

これまで、一般的なブログ作成サービスについて紹介してきましたが、「note」は一般的なブログとは少し違います。

もちろん一般的なブログのように利用することも可能ですが、最も大きなポイントはあなたの書いた記事を有料記事として公開することができるという点です。

そんなnoteのメリットはこちらです。

稼げるライティングの考え方

では早速、ブログやアフィリエイトで稼げるライティングをできる様になるための考え方から解説していきます。

ここは本当に、本当に重要なことなので記事を書き始める前に、必ず頭に入れておいてください。

読者はとにかく読みたくない

いきなりですが、大前提として「あなたの記事は読んでもらえない」ことを理解してください。

この記事の内容と反対のことを言っているように感じるかもしれませんが、これを理解することがブログやアフィリエイトで稼げる様になるための第一ステップです。

「読み物」には２つの種類があります。

  1. 読むこと自体が楽しいもの
  2. 情報収集

①はマンガや小説など、文章を読むこと自体が目的で、読むこと自体に楽しみがあるものです。

ブログの場合は

  • 自分がファンである有名人のブログ
  • 読み物として面白い人のブログ

でも大半の人はそうじゃないですよね。なのでほとんどの方のライティングは②の情報収集を目的とした方向けのものになります。

言うまでもなく、WEB上の記事の大半は、②の情報収集にあたります。

②は文章自体を読みたいわけではなく、「何かわからないことを調べている」ものです。

情報収集の場合、「知りたいことを理解」することが目的なので、とりあえず早く答えを知りたいというニーズになります。

情報収集の場合は、答えさえ知れれば、ウェブの記事でも、友達のアドバイスでも、youtubeの動画でもなんでもいいですよね。

そして情報収集をするときに、必ず全員が考えていることは「早く知りたい」ということです。

つまり「時間をかけて文章を読みたくない」というのが大前提になるわけです。

要するにこの記事を読んでいるあなたも、僕の話を聞きたいわけではなく「稼げるライティングの書き方」さえ知れればそれで良いわけですね。

それでも「なげーよ」って思う方もいると思いますが、今できる範囲で出来るだけ短く早く伝えようと努力していることだけはご理解いただければと思います。

滞在時間が大事？

じゃあ滞在時間が短くなることは問題ないの？という疑問が生まれると思います。

結論としては、OKな場合とNGな場合があります。

滞在時間が短くなって良い場合と悪い場合

OKなケース
あなたの記事を読んで、またはあなたが紹介した記事に移動して、検索が終了した場合。

NGなケース
あなたの記事から離脱して検索画面に戻った場合。

これがSEO的な重要ポイントになります。

SEOは奥が深いので、解説動画を小出しで出していきますが、現時点で理解しておいて欲しいのは、

SEOで最もマイナス評価を受けるのは、「読者が記事から検索画面に戻った場合」だということです。

Googleの考えでは「検索を終了した（ブラウザを閉じた）　＝　読者が疑問を解決できた」と定義しています。

なので検索画面に戻った場合、「その記事は読者の悩みを解決できなかった　＝　役に立たなかった」という判定をするわけです。

つまり、滞在時間がいくら長くても、読者が検索画面に戻ったらマイナス評価なんです。

逆に滞在時間が短かったとしても、「検索行動を終了した　＝　読者は悩みを解決した」ということでプラス評価を得ます。

逆に話を引っ張ったり、結論がなかなかわからない長文の記事は、読者の離脱を招きます。滞在時間がどうとかいう次元じゃないマイナス評価を受けるわけです。

SEOを意識したライティングでも、一番重要視するべきなのは「速く疑問を解決できる」こと。

結局、結論を先に書くのが正解なんです。

「検索意図」という概念

次に理解しておくべきなのが、「検索意図」です。

「検索意図」とは何かというと、「読者さんが知りたいこと」です。

先ほど解説した通り、私たちが書くWEBライティングの大半は「情報収集」をしている読者さんのためのものです。

なので検索意図に対して、あなたの記事がずれていると、読者さんにとってなんの解決にもならない記事になってしまうわけです。

「そんなことねーだろ」って思うかもしれないですが、WEBライティングの世界では非常によくあることです。

例えば、

「アップルウォッチ 価格」

というキーワードに対して記事を書いているのに、なぜか記事の冒頭に

「アップルウォッチ とは、iPhoneで有名なアップル社が作った時計型のデバイスです」

みたいな解説を入れちゃうパターンがよくあります。

これを現実での会話にしてみると、

読者さん「アップルウォッチ の料金っていくらくらい？」

筆者「アップルウォッチ というのは、iPhoneで有名なアップルが作っている時計型デバイスなんだよ！

特徴としては…

スペック的には…

で、気になる料金なんだけど５万円くらいからだから、Apple公式サイトを見てみてね♪」

みたいな会話になっているわけです。

ウザすぎますよね笑

WEBライティングの難しいところは、この「検索意図」を「検索キーワード」から判断しなければいけないという点です。

具体的なテクニックは中盤で解説しますが、現段階では「読者の検索意図を正確に理解しないといけない」という点を肝に銘じておきましょう。

読まれるライティングの記事構成

次に具体的に、ライティングのテクニックをご紹介していきます。

記事は本文からライティングする

まず大前提ですが、記事は本文から書いていきます。

そして最後に、導入文とタイトルをつけてください。

記事が読まれるかどうかは、冒頭の導入文に全てがかかっていると言っても過言ではありません。

なので本文とまとめを完成させてから、

  • 「この記事ではこんなことを解説しました」
  • 「この記事を読むと、このようなことが理解できます」

という導入文を書くようにしましょう。

導入文には必ず「記事を読むメリット」を入れる

記事の中で最も離脱（読者が去ること）が多い箇所は、導入文です。

そして導入文には必ず「読むメリット」…厳密には読む人にとってのベネフィットを入れる必要があります。

つまり記事の最初で「この記事を読んだらどんな良いことがあるか」をハッキリと伝えないといけない訳です。

これはブログやアフィリエイトで月5万を稼ぐためには必須のスキルです。

読者は記事に入ってきてから一瞬で、その記事を読むか別の記事を探すかを判断します。

1秒くらいしか猶予はないと思ってください。

つまり導入文で読む気になってもらわないと、その後の文がいくら良くても無意味になる訳です。

なので記事の本文が完成してからの方が良い訳です。

記事ができていないのに導入文を書くのは、商品が完成していないのにCMを考えるようなものです。

実際に商品が完成して初めて、その商品のアピールできる良さがわかります。

なので本文とまとめができてから、導入文を書くのが基本です。

また、記事を書いていると、予定していなかった情報を追加したり、内容を修正することがあります。

先に導入文から始めると、２度手間になることもあるので、そう言った意味でも最後に書くようにしてください。

この記事でも、一番最初に読んだ後に得られるベネフィットを書いていますね。

これが記事最初で離脱を防ぐためのライティングになります。

見出し、結論、補足の順番

何度も言っている通り、とにかく記事は最短時間で結論がわかることが重要です。

なので、ライティングするときには飛ばし読みをしながらでも内容が理解できるように意識することが大切です。

飛ばし読みで内容を理解できるようにするには、リズムが一定であることが重要です。

ということで、目指す記事構成は、

  1. 見出し
  2. 結論
  3. 補足説明

というリズムを一定にしたライティングです。

これだと読者さんは、

  • 基本的に見出しの周辺だけを見る
  • 詳しく知りたい時だけ、根拠の部分を熟読する

という読み方が可能になります。

なので最短時間で内容を理解することができます。

逆にリズムがバラバラで、結論が書いている場所が毎回違うと、記事全体を熟読しないと内容が理解できません。

要するに、読者さんが知りたい結論がどこに書いているか探しながら読まないといけないわけです。

もちろん完全にリズムを一定にするのは現実的に不可能なこともあります。

ですが、出来るだけ見出し周辺に結論をまとめるという意識で書いた方が、読者さんが読みやすいライティングになるので、意識しておきましょう。

ちなみにですが、写真や画像が出てくる位置も可能な限り、一定にした方が読みやすいです。

見出し直下に必ず画像を入れる…など。

一文は３０文字以内

初心者の方に多いのが、一文が長いライティングです。

文は長ければ長いほど、理解するのが難しくなります。

気を抜くと僕も1文を書いてしまいがちですが、できるだけ３０文字以内に抑えることを意識しています。

１文が長くなった場合は、できるだけ２文に分けるように意識しましょう。

（７３文字）

良い例

このようにかなり意識しているつもりの僕でも、うっかりやってしまいます。

なので常に頭に入れておく必要があります。

具体的な見出しの作り方（検索意図を理解する）

ライティングの大枠を理解したところで、次に具体的な見出しの作り方（記事構成の構築）について解説します。

ここでは記事のテーマ（狙うキーワード）が決まっているという前提でお話を進めていきます。

狙うキーワードの見つけ方は、サイト設計の話になってくるので、また別の記事で解説します。

ライティングの基本：検索上位１０記事の見出しから作る

まずライティングしていく記事の見出しは、検索上位１０サイトの中から選ぶのが基本戦略になります。

具体的にあなたが作る記事のテーマ（狙っている検索キーワード）で検索してみましょう。

そしてその上位１０記事を見て、見出しのテーマをリストアップしてください。

ちなみにここでは、メーカーのウェブサイトや行政のサイトなど、ブログ・アフィリエイトサイトとかけ離れているものは対象から外してOKです。

これをしっかり見ていくと、

  • ライティングする記事テーマで絶対に外せない内容
  • 上位サイトにあまり書いていないけど、読者が知りたそうなもの

が見つかります。

それをもとに記事の見出しを考えていきます。

また、この段階で「検索意図」がわかります。

例えば「アップルウォッチ」で検索した場合、大半の記事に「最新機種と過去機種の違い」についての見出しが入っています。

つまり、アップルウォッチ で検索している人の多くは、「最新機種と過去機種の違い」を知りたがっている。

検索意図の一つは「機種の比較」であることがわかります。

このように、検索結果に出てくる記事から、読者が知りたがっていること（検索意図）がわかるわけです。

関連キーワードから、不足している情報を探す

上位１０記事の見出しをチェックした後は、関連ワードを調べます。

僕の場合は、

  • Googleキーワードプランナー

記事を書く順番

  2. まとめ
  3. 導入文
  4. 記事タイトル

導入文には必ず「記事を読むメリット」を入れる

記事の中で最も離脱（読者が去ること）が多い箇所は、導入文です。

読者は記事に入ってきてから一瞬で、その記事を読むか別の記事を探すかを判断します。

1秒くらいしか猶予はないと思ってください。

つまり導入文で読む気になってもらわないと、その後の文がいくら良くても無意味になる訳です。

なので記事の本文が完成してからの方が良い訳です。

記事を書いていると、予定していなかった情報を追加したり、内容を修正することがあります。

なので記事が出来上がってから「この記事はこんなことについて書きました！読んでくれたらこんなメリットがあります！」という形で書くのが正解です。

この記事でも、一番最初に読んだ後に得られるベネフィットを書いていますね。

これが記事最初で離脱を防ぐためのライティングになります。

少し細かい話になりますが、導入文に力を入れるのはアクセスを集めるためにも非常に重要です。

記事を上位表示させるためのSEOは非常に奥が深いので、ここで全てを話すことはできませんが、現段階で１つだけ理解しておいてください。

それは記事冒頭での読者の離脱は、SEOで最もマイナス評価を受けることの一つだということです。

一瞬で離脱された記事＝その人にとって全く役に立たなかった

ということです。

  1. 見出し
  2. 結論
  3. 根拠

結論が後に書いていると、最後まで読まないと内容が理解できないので時間がかかります。

まず結論から書くことを意識しましょう。

１文はできるだけ短く

初心者にありがちな読みにくい記事の典型的なパターンは１文が長いことです。

これは読者にとってかなり読みにくい記事なので、離脱率が高くなってしまいます。

僕は１文を３０文字以内を推奨しています。

さらに、ここでも結論を先に書くことも意識したほうがベターです。

英語の文章をイメージするとわかりやすいかもしれません。

例えば、日本っぽい文で書くと

  • 僕は友達と旅行先でこってりしたラーメンを食べました。

ですが、英語っぽく

  • 僕はラーメンを食べました。それは友達と旅行先でこってりしたものです。

ちょっと極端ですが、これをイメージしておくと文の冒頭に結論が来るので、WEB記事としてはベターです。

この書き方にするとスマホをスクロールしながら飛ばし読みしても、ざっくりと内容を理解できるので読みやすいわけです。

直感的な操作で簡単にブログを作るならWiX

WiXは、デザインやプログラミングの知識がなくても、マウスのドラッグ＆ドロップの簡単な操作だけでブログを作ることができます。

2006年から開始されたサービスで、今では世界190か国で1億6,000万人以上の利用者がいるなど、実績も高く利用者からの評判も良いです。

また、WiXは

  • ブログ維持のコストがかからない
  • スマホのデザインの対応も可能
  • 500種類以上のテンプレートから選んでHP制作を始められる
  • 2万点以上の画像や動画素材を無料で利用できる

など、無料プランとは思えないほど充分な機能がついています。

”ブログ専用”のテンプレートも用意されているため、より簡単にブログの作成が始められます。

他のソフトだと「無料期間が終わると、作成したHPが見れなくなってしまう」ということがありますが、WiXだと無料で使い続けることができます。

1つのアカウントで複数のブログを管理するならSeesaa

Seesaaブログの最大の特長は1つのアカウントで5つまでブログを管理することができる点でしょう。

ほとんどのサービスは1アカウントにつき1つのブログなので、別のブログを作成する場合には違うアカウントを取得しなければなりません。

ですから、いくつかのブログを運営したいという場合にはSeesaaがおすすめです。

しかも、2016年には利用可能な容量も2GBから5GBに変更されたので、高解像度の画像も容量を気にせず置くこともできます。

アフィリエイトに関しても特に制限がなく、グーグルアドセンス（Google AdSense）の設置ができるのもポイントです。

ブログ開設で最も重要なのがコンテンツの作り方です。

ここではブログの作り方ということで、ブログの内容について考えてみましょう。

あなたのブログなので、何でも好きなことを書けば良いとは言うものの、ただの日記では勿体ない。何故なら、日常のことを書いて読者に喜ばれるのは芸能人ブログだけだからです。

ですから、多くのファンを作れる有益な情報を発信していく必要があります。その為には次のような手順で考えるのがおすすめです。

  1. ブログのテーマとコンセプトを決める
  2. ブログの名前を決める
  3. ブログのデザインを決める
  4. 記事を作成する

上記のような手順を踏み、それに従って記事を積み重ねることで、ブログのブランディングもできます。

ブログの作り方！テーマとコンセプトを決めよう！

実は、ほとんどの悩み『HARMの法則』によって分類できると言われています。HARMとは次の4つの悩みの頭文字を取ったものです。

  • Health：健康・医療・美容
  • Ambition：夢・希望・願望
  • Relation：人間関係・恋愛
  • Money：お金・資産

あなたの悩みも上記の4つの中にあるのではないでしょうか。

HARMに分類される悩みというのは、実はお金を払ってでも解決したいという深い悩みが多くなっています。

ですから、あなたがブログを開設する時のテーマとしては上記に当てはまるテーマに沿った内容がおすすめです。

ただし、あなたの全く興味のない分野ではブログを続けることが難しいので、興味のある分野の悩みを取り上げるのが良いでしょう。

キャッチーなブログの名前を決めよう！

ブログの名前というのはとても大事です。特にリピーターを獲得する為にもキャッチーなブログ名にしましょう。

一度聞いたら忘れないようなブログ名の場合、そのブログ名で検索されるようになります。すると、検索エンジンからも認識されるのでSEOでも有利になります。

ブログのテーマ・コンセプトが決まったら、ブログ名もそのテーマやコンセプトを表しているようなブログ名になっていると言うこと無し。

ブログ名は後からでも変更可能ですが、ある程度認識されてから変更するのはおすすめできません。できれば最初に決めておいてください。

ブログのデザインを決める

ブログの作り方を考える上で、ブログのデザインも大切です。無料ブログではデザインの選択肢が少なく、自分でカスタマイズするのも難しい場合があります。

その点、デザインの自由度が高いのがWordPressです。

WordPressはテーマの数も非常に多く、デザインに拘っているテーマも数多くあります。

もし、あなたがデザインのセンスが無いと思う場合には有料テーマを使用されることをおすすめします。

ブログの作り方で最も重要なのがコンテンツです。

また、記事の内容についても前述した通り、単なる日記ブログではなく、悩みを解決するような内容が好ましいでしょう。

購入するのは独自ドメイン

ドメインとは何かというと、インターネット上の住所を解りやすく表していると思ってください。

インターネット上の住所は本当はIPアドレスという数字の羅列で表わされます。

しかし、数字の羅列では非常に解り辛いので、IPアドレスを文字列に置き換えて利用します。文字列は数字とは違い、意味の表現もできますし、覚えやすいという利点もあります。

ですから、住所は人間に解りやすい文字で表現し、通常はこちらを使用します。つまり、コンピュータでは数値に置き換えて使用しているようなイメージ。

では、具体的にドメインとはどのような仕組みになっているのでしょうか。実際はいくつかのパートに分けられています。

例えば、弊社の「https://www.mixhost.jp/」というドメインを見てみると、

  • https://
  • www.
  • mixhost
  • .jp/

という3つのパートが組み合わさっています。

実際は「www.」は省略され「https://mixhost.jp/」となっています。

この中で、あなたが自由に決められるのは「mixhost」に相当する部分。

そして「.jp」に相当する部分は予め決められた文字列の中から選択することになります。

「.jp」に相当する部分のことを「トップレベルドメイン」と言い、中には個人では購入できないものやその国に住所がなければ購入できないものもあります。

では、トップレベルドメインについてもう少し詳しく見て行きましょう。

トップレベルドメインとは

トップレベルドメインには、大きく3種類に分けられます。

  • 分野別トップレベルドメインgTLD
  • 国別トップレベルドメインccTLD
  • インターネットインフラ用のTLD

上記の中で、「インターネットインフラ用のTLD」はユーザー登録のできないドメインです。ですから、ユーザーが登録できるのは上記の2種類ということになります。

分野別トップレベルドメイン

分野別トップレベルドメインはgTLD（Generic Top Level Domain）と呼ばれるドメインです。

gTLDには次のようなドメインがあります。

  • .com
  • .net
  • .com
  • .org
  • .info

その他にも多数ありますが、分野・使い道で分類されるイメージです。

しかし、制限の無い場合が多く、空いていれば自由に購入することができるドメインが多数あります。

また、gTLDの中にはsTLD(Sponsored Top Level Domain)と呼ばれるスポンサーを持つドメインもあります。

国別トップレベルドメインccTLD

国別トップレベルドメインはccTLD（Country code top-level domain）と呼ばれるドメインで、国や地域ごとに割り当てられています。一部を下記に記載します。

  • .jp：日本
  • .us：アメリカ合衆国
  • .uk：イギリス
  • .it：イタリア
  • .eu：EU連合
  • .cn：中国

あなたが日本に住んでいるのなら、基本的に「.jp」以外のccTLDを購入することはできません。しかし、一定の条件を満たせば他国のccTLDを購入することもできる場合があります。

ドメイン取得サービスの選び方

ここまででドメイン取得サービスについての意味や専門用語、おすすめのドメイン取得サービスをご紹介しました。

では、ドメイン取得サービスの選び方のポイントはどんな所にあるのでしょうか。

ここでは大きく3つのポイントをご紹介します。

初期費用が安いドメインを選ぶ

当たり前ではありますが、初期費用が安いドメインを選ぶのが良いです。

ドメイン取得サービスの初期費用が安い所もそうですが、ドメインの種類によっても価格が違うので注意が必要です。

高いものだと1万円のドメインも存在しますが、その多くは「企業」「団体」などが利用するもので個人レベルでは価値が薄いものです。

ですので、個人の場合であれば1000円前後のドメインから始めるのが良いでしょう。

特典が魅力的な所を選ぶ

各ドメイン取得サービスには特典が用意されていて「指定レンタルサーバーの初期費用割引」や「新規取得1円から」、「ポイント贈呈」などのキャンペーンが存在します。

mixhostでは、レンタルサーバーを同時契約（契約期間が12ヶ月以上の場合）には、対象ドメインに関して、「取得するドメイン費用が永久無料」のキャンペーンが受けられます。

バリュードメインは「1つ目のドメイン取得0円」の特典を長年行っています。

また、お名前.comやエックスドメインも「新規取得1円」のキャンペーンがあるので、気になる所をチェックしてみましょう。

ドメイン取得サービスの選び方

  1. 初期費用が安いドメインを選ぶ
  2. 特典が魅力的な所を選ぶ

 

ドメイン取得サービスを選ぶ際の注意点

ドメイン取得サービスを選ぶ際の注意点はどんな所にあるのでしょうか。

2つの注意点としてご紹介します。

年間更新費用の違いに注意

ドメイン取得料金（初期費用）の安さばかりに目が移ってしまいがちですが、「年間更新の費用」にも違いがあるので注意が必要です。

ドメイン取得サービスもそうですが、ドメインの種類でも数千円の違いが出てくるので注意しましょう。

おすすめのドメイン取得サービスで紹介しているものであれば、年間更新の費用が安いのは「エックスドメイン（1,298円）」「バリュードメイン（1,287円）」辺りになります。

mixhostでは、レンタルサーバーを同時契約（契約期間が12ヶ月以上の場合）には、対象ドメインに関して、「ドメインの年間更新の費用が永久無料」のキャンペーンが受けられます。

ドメイン取得サービスのスペックや運営停止に関しては「ドメイン移管」で対応できるので、そこまで気にする必要はないと思います。

 

選んだドメインで後悔しないように注意

ドメイン取得サービスでドメイン取得をする上で、1番の注意点となるのが「ドメインのネームバリュー」や「ドメイン名の不満」などの選んだドメインの後悔です。

「.comにすれば良かった」「もう少し短いドメイン名にすれば良かったかな」などはよくある後悔です。

ですので、ドメイン取得サービスで新規取得が安いからと言って、全く覚えにくいドメインを取得するのは後悔の元です。

最終的には決断する必要がありますが、しっかりと吟味して「ユーザーが覚えやすいドメイン名」を選ぶ事をおすすめします。
